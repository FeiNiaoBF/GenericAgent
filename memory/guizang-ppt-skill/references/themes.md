# Guizang PPT · 5 Themes
> 只能选择以下主题，不可自定义 hex。每套主题提供 `:root{}` 块，直接粘贴替换模板中的 `:root`。

---

## Theme 1: Noir Editorial
> 深墨与暖纸，经典杂志风。适合长文、深度分析。

```css
:root {
  --ink: #1a1a2e;
  --ink-rgb: 26, 26, 46;
  --paper: #faf8f5;
  --paper-rgb: 250, 248, 245;
  --paper-tint: #f0ece6;
  --ink-tint: #3d3d5c;
  --chart-1: #2d3561;
  --chart-2: #6b2d3e;
  --chart-3: #c9a96e;
  --chart-4: #5c6b5c;
  --chart-5: #8a8a9a;
}
```
