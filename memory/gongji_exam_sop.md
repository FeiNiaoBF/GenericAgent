# 公基备考 SOP (唧式陪练) (v1.0)
> 触发: 公基/备考/刷题/时政/常识判断/事业单位考试
> 依赖: web_search_sop | vault_knowledge_sop | user_profile_sop

## 执行摘要（≥1步执行前必读）
1. 路由：具体题→答疑 | 无题→随机出题 | 累了→收尾统计+鼓励
2. 按模块出题→连对3题升难度→错题归档标记 #公基+#错题
3. 每20题提醒复习错题→模拟考试10/20/30题限时→结束输出得分率+薄弱分析 → 🛑 过验证门禁

## 0. 路由合约
```
主人说"学公基" →
├─ 有具体题 → Phase 2 答疑

```
skip: 贴题直接问答(跳过出题) | 上轮同类换模块 | 连对3题升难度

## MOC创建规范（★实操必读）
### 两种MOC的区别
| | 顶层MOC | 主题MOC |
|---|---------|---------|
| 模板 | 从"01-05Cat·LLM版.md"起步 | 从"MOC.md"(v3.0统一模板)起步 |
| 定位 | 领域领航(5大分部) | 单一知识主题聚合 |
| 指向 | 指向子MOC `[[子MOC名]]` | 指向笔记 `[[笔记名]]` |
| 区块 | domain/subjects/顶层关系/增长引擎/7区地图 | v3.0按需启用(地图边界/精选导航/知识断言/动态索引等) |
| 字段 | 无moc字段 | `moc: "[[所属顶层MOC]]"` |

### 创建主题MOC步骤
1. **确认模板**: 用 `99.System/Templates/MOC.md` (v3.0)，禁用顶层MOC模板
2. **填写YAML**: `type: moc | moc: "[[所属顶层MOC]]" | status: seedling`
3. **按需启用区块**: 地图边界(scope+核心问题≥2) → 精选导航(≥3篇) → 知识断言 → 动态索引(Dataview)
4. **链接回顶层MOC**: 顶层MOC中新增 `- [[主题MOC名]]` 精选导航项
5. **同步Quests三件套**:
   - `公基·Dashboard.md`: 更新对应模块进度→🟡进行中 + MOC链接
   - `公基备考.md`: updated日期 + Step 2✅ + 今日行动标记
   - `公基备考地图.md`: updated日期
6. **验证门禁**: MOC有scope? 核心问题≥2? 顶层MOC已接入? Quests三件套已同步?

### 禁手
🚫 主题MOC禁用顶层MOC模板(含domain/subjects/7-section结构)
🚫 创建MOC后不同步Quests三件套
🚫 顶层MOC不更新子MOC入口

## Phase 1: 出题模式
知识模块轮换: 法律→政治→经济→文史→科技→地理→时政→管理
规范: 单选为主(4选项) | 难度动态调整 | 优先真题改编 | 每5题错题统计
Guards: 不连续同模块 | 不出未验证题 | 答对→简短确认 | 答错→解析+归档

## Phase 2: 答疑模式
流程: 读题(确认题型+模块)→搜索(web_search_sop, 政府>教材>百科)→解析(选项分析+陷阱+延伸)→**三步归档**（见下方）
格式: 📋题目 → ✅答案 → 📝逐项解析 → 💡延伸

### ⭐ 错题三步归档流程（★收到错题必执行，顺序不可跳过）

**Step 1: 写知识笔记**（先理解再记忆）
1. 从错题提取核心知识点（如"宪法修正案程序"而非整道题）
2. 判断所属模块→查资料→用自己话写Wiki风原子笔记
3. 归档到对应目录（见上方"笔记归档策略"映射表）
4. frontmatter 必填: `tags: [公基]`, `moc: "[[对应主题MOC]]"`, `status: seedling`, `created: YYYY-MM-DD`
5. 笔记正文: 定义→思想史/来源→逻辑展开→应用场景→关联（禁"例题/陷阱"节）

**Step 2: 更新MOC**
1. 确认该知识点的主题MOC存在（如`MOC｜法律-宪法.md`）
2. MOC不存在→按vault_knowledge_sop §5创建主题MOC（用MOC.md v3.0模板，moc字段指向顶层MOC）
3. MOC存在→检查Dataview索引是否能自动发现新笔记（靠moc字段）
4. 新建MOC时同步Quests三件套（Dashboard/备考/备考地图）

**Step 3: 生成Anki卡片**
1. 将错题完整内容追加到该知识笔记的 `## 🐛 错题` section
2. 运行 `python memory/anki_export.py` 生成 `.apkg`
3. 告知主人卡片数量+笔记路径

**三步流程示意**:
```
主人发错题 →
  ① 提取知识点 → 写 [[宪法修正案程序]].md → 03.Library/Notes/Law/
  ② 确认 [[MOC｜法律-宪法]] 存在 → moc字段自动挂载
  ③ 错题追加到笔记 → anki_export.py → .apkg
```

## Phase 3: 错题复习
触发: 主人说"复习错题" 或 每20题提醒
流程: Dataview调取 #公基+#错题 → 抽3-5题重测 → 二次错→重点标记 | 连对→移除

## Phase 4: 模拟考试
触发: 主人说"来套模拟卷"
10/20/30题可选 | 限时1题/分 | 模块分布模拟真实比例 | 结束: 得分率+薄弱分析+建议
