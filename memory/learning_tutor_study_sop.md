# 学习助教 · learning_tutor_study_sop

## 入口路由
- 英语：留在本SOP执行。
- CS/编程：跳 `programming_teaching_sop.md`；本SOP只做路线与节奏管理。
- 周末主动追踪：读了几章/几节？新词/断言数？听力/习题次数？最困难处？

## 英语分支
主人画像：通勤/洗碗适合音频，晚间约1h15；偏 `Acquisition > Learning` 沉浸式；Tom Sawyer主、Robinson Crusoe副。

### 三轨
- Fiction：Tom Sawyer(~950L)主 / Robinson Crusoe(~1200L)副，比例3:1；audiobook → 阅读 → 每章≤5生词。
- Info：Tech blogs / BBC News / Medium。
- Listening：通勤 BBC 6min English + Techmeme；洗碗 audiobook；晚间预热3–5min。
- 主人音频：先确认音频有效 → 转写 → 反馈“原句/更自然表达/原因/可复用chunk/今日复练句”；只挑1–3个高价值点入库。

### 词汇与节奏
字段：`source / etymology / collocation / mnemonic / review_status(0→1→2→3)`。
SRS：当天 → 次日 → 第7天 → 第30天。
里程碑：M1-2 Tom Sawyer+200词；M3-4 Robinson Crusoe+500词；M5-6 3本书+800词+50字英文日记。
避坑：不同时读3本以上；词汇复习≤15min/天；未触发不强聊英语；卡顿降级Tom Sawyer。

## 日常职责
- 早上通勤：提醒播客/简单英语问答。
- 晚间：打开笔记 → 引导阅读/学习 → 记录词汇或断言。
- 每天日记：把英语任务追加到当日日记 `今日任务`/`晚上` 待办区，不写入 `🐣 唧の足迹`。
- 标准日：Listening 5–10min + Fiction 1–2页且≤3生词 + Output英文1句 + Review昨日1–3表达。
- 忙碌日：降级为保底输入10min + 1表达。
