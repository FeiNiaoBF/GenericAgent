# HTML PPT Skill - Recommended Templates

> 只保留唧前面推荐给主人使用的高价值模板。

## CS 教学页核心模板

| 优先级 | Template | 用途 |
|---:|---|---|
| 1 | `two-column.html` | 概念 + 示例，教学万能结构 |
| 2 | `code.html` | 代码高亮，编程教学核心 |
| 3 | `flow-diagram.html` | 执行流程、请求链路、事件循环、算法步骤 |
| 4 | `arch-diagram.html` | 系统架构、三层结构、网络模型 |
| 5 | `comparison.html` | JS vs TS、REST vs GraphQL 等对比主题 |
| 6 | `diff.html` | 展示代码变化、类型系统收益、重构前后 |
| 7 | `terminal.html` | CLI、构建、运行结果演示 |
| 8 | `table.html` | 语法、特性、复杂度、取舍对比 |
| 9 | `process-steps.html` | 算法流程、工程流程、分步骤解释 |
| 10 | `mindmap.html` | 课程总览、知识地图 |
| 11 | `timeline.html` | 技术演化史、版本发展 |
| 12 | `pros-cons.html` | 优缺点总结页 |

## 完整 PPT 骨架模板

推荐顺序：

1. `cover.html`
2. `toc.html`
3. `section-divider.html`
4. `two-column.html`
5. `code.html`
6. `flow-diagram.html`
7. `comparison.html`
8. `table.html`
9. `arch-diagram.html`
10. `process-steps.html`
11. `cta.html` 或 `thanks.html`

## 默认结尾

- 正式汇报：`cta.html`
- 课堂 / 分享：`thanks.html`
